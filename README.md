🔍 Network Scanning Automation Toolkit

📌 Description
This project automates network scanning using Python along with powerful system tools:

Ping (Host Discovery)

ARP (Network Mapping)

Nmap (Port & Service Scanning)

🛠️ Installation

Install Python

python --version

Install Nmap

Linux:

sudo apt install nmap

Windows: https://nmap.org/download.html

▶️ Usage

Ping Scanner

python ping_scanner.py

ARP Scanner

python arp_scanner.py

Nmap Scanner

python nmap_scanner.py

⚠️ Disclaimer
This tool is intended for educational purposes only. Use it strictly on authorized networks.
